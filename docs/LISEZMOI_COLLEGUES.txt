Markdown Converter — installation rapide (macOS)
==================================================

Contenu du ZIP :

- « Markdown Converter.app » : l’application (double-clic pour lancer).
- LISEZMOI.txt : ce fichier (copie de la documentation courte).

Installation recommandée :

1. Décompressez le ZIP.
2. Glissez « Markdown Converter.app » dans le dossier « Applications » de votre Mac (recommandé).
3. Au premier lancement, si macOS affiche un avertissement (développeur non identifié), faites un clic droit sur l’icône puis « Ouvrir », ou autorisez dans Réglages système → Confidentialité et sécurité.

Pandoc n’est pas inclus dans l’application : le secours Pandoc ne fonctionne que si Pandoc est installé séparément sur votre Mac. L’application fonctionne sans.

Pour toute question technique, consultez le dépôt GitHub du projet (README et issues).
